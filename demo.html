How do I configure my router?
To configure your router, connect to the admin panel by entering 192.168.1.1 in your browser.
Log in using your admin credentials, then navigate to the network settings section.
Answer sourced from Router Manual (PDF)
How can I safely restart the router?
To safely restart the router, use the reboot option in the admin panel.
Avoid unplugging the power cable while the router is running to prevent configuration damage.
Source: Page 18, Router Manual PDF
